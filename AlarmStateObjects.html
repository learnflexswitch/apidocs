AlarmState Object¶

state/Alarm¶

  • Multiple objects of this type can exist in a system.
PARAMETER NAME DATA TYPE DESCRIPTION DEFAULT VALID VALUES
SrcObjName [KEY] string Alarm event name picked up from events.json N/A N/A
EventName [KEY] string Alarm event name picked up from events.json N/A N/A
OwnerName [KEY] string Alarm owner daemon name picked up from events.json N/A N/A
OwnerId [KEY] int32 Alarm owner daemon Id picked up from events.json N/A N/A
EventId [KEY] int32 Alarm event id picked up from events.json N/A N/A
ResolutionTime string Resolution Time stamp N/A N/A
SrcObjKey string Fault Object Key N/A N/A
OccuranceTime string Timestamp at which fault occured N/A N/A
ResolutionReason string Cleared/Disabled N/A N/A
Severity string Alarm Severity N/A N/A
SrcObjUUID string Fault Object UUID N/A N/A
Description string Description explaining the fault N/A N/A

FlexSwitch CURL API Supported¶

  • GET By Key

    curl -X GET -H ‘Content-Type: application/json’ –header ‘Accept: application/json’ -d ‘{<Model Object as json-Data>}’ http://device-management-IP:8080/public/v1/state/Alarm

  • GET ALL

    curl -X GET http://device-management-IP:8080/public/v1/state/Alarms?CurrentMarker=<x>&Count=<y>

  • GET By ID

    curl -X GET http://device-management-IP:8080/public/v1/config/AlarmState/<uuid>
