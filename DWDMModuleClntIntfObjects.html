DWDMModuleClntIntf Object¶

config/DWDMModuleClntIntf¶

  • Multiple objects of this type can exist in a system.
PARAMETER NAME DATA TYPE DESCRIPTION DEFAULT VALID VALUES
ModuleId [KEY] uint8 DWDM Module identifier N/A N/A
ClntIntfId [KEY] uint8 DWDM Module client interface identifier N/A N/A
HostRxSerializerTap2Delay uint8 Host RX Serializer tap 2 control 5 N/A
HostTxEqLfCtle uint8 Host interface TX deserializer equalization. LELPZRC LF-CTLE LFPZ gain code. 0 N/A
TXFECDecDisable bool 802.3bj FEC decoder enable/disable state for traffic from Host to DWDM Module false N/A
EnableIntSerdesNWLoopback bool Enable/Disable serdes internal loopback false N/A
HostRxSerializerTap1Gain uint8 Host RX Serializer tap 1 control 7 N/A
HostRxSerializerTap2Gain uint8 Host RX Serializer tap 2 control 15 N/A
HostTxEqCtle uint8 Host interface TX deserializer equalization. LELRC CTLE LE gain code. 18 N/A
RXFECDecDisable bool 802.3bj FEC decoder enable/disable state for traffic from DWDM module to Host false N/A
RxPRBSPattern string RX PRBS generator pattern 2^31 2^7, 2^15, 2^23, 2^31
AdminState string Administrative state of this client interface UP UP, DOWN
EnableHostLoopback bool Enable/Disable loopback on all host lanes of this client interface false N/A
EnableRxPRBS bool Enable/Disable RX PRBS generation for all lanes of this client interface false N/A
EnableTxPRBSChecker bool Enable/Disable TX PRBS checker for all lanes of this client interface false N/A
HostTxEqDfe uint8 Host interface TX deserializer equalization. s-DFE 0 N/A
HostRxSerializerTap0Delay uint8 Host RX Serializer tap 0 control 7 N/A
HostRxSerializerTap0Gain uint8 Host RX Serializer tap 0 control 7 N/A
NwLaneTributaryToClntIntfMap uint8 Network lane/tributary id to map to client interface N/A N/A
TxPRBSPattern string PRBS pattern to use for checker 2^31 2^7, 2^15, 2^23, 2^31
