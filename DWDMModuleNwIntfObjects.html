DWDMModuleNwIntf Object¶

config/DWDMModuleNwIntf¶

  • Multiple objects of this type can exist in a system.
PARAMETER NAME DATA TYPE DESCRIPTION DEFAULT VALID VALUES
ModuleId [KEY] uint8 DWDM Module identifier N/A N/A
NwIntfId [KEY] uint8 DWDM Module network interface identifier N/A N/A
ClntIntfIdToTributary0Map uint8 Client interface ID to map to network interface tributary 0 N/A N/A
ClntIntfIdToTributary1Map uint8 Client interface ID to map to network interface tributary 1 N/A N/A
ModulationFmt string Modulation format to use for this network interface 16QAM QPSK, 8QAM, 16QAM
RxPRBSPattern string PRBS pattern to use for checker 2^31 2^7, 2^15, 2^23, 2^31
AdminState string Administrative state of this network interface UP UP, DOWN
RxPRBSInvertPattern bool Check against inverted PRBS polynomial pattern true N/A
TxPower float64 Transmit output power for this network interface in dBm 0 N/A
TxPowerRampdBmPerSec float64 Rate of change of tx power on this network interface 1 N/A
TxPulseShapeFltrRollOff float64 TX pulse shape filter roll off factor 0.301 N/A
FECMode string DWDM Module network interface FEC mode 15%SDFEC 15%SDFEC, 15%OvrHeadSDFEC, 25%OvrHeadSDFEC
DiffEncoding bool Control to enable/disable DWDM Module network interface encoding type true N/A
EnableTxPRBS bool Enable TX PRBS generation on this network interface false N/A
TxPulseShapeFltrType string TX pulse shaping filter type RootRaisedCos RootRaisedCos, RaisedCos, Gaussian
ChannelNumber uint8 TX Channel number to use for this network interface 48 N/A
TxPRBSInvertPattern bool Generate inverted PRBS polynomial pattern true N/A
TxPRBSPattern string Pattern to use for TX PRBS generation 2^31 2^7, 2^15, 2^23, 2^31
EnableRxPRBSChecker bool Enable RX PRBS checker false N/A
